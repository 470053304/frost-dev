SymFunction.SymFunction (   varargin)

The class constructor function.

Parameters
vararginvariable nama-value pair input arguments, in detail:
SymFunction ( Name, Expression, DepSymbols, ParSymbols, PreCommands )
Required Parameters for varargin:
  • Name the name of the function.
  • Expression the symbolic expression of the function
  • DepSymbols the symbolic representation of variables in the expression
  • ParSymbols the symbolic representation of constants in the expression
  • PreCommands a list of commands requires to be executed prior to the expression
