Detailed Description

HZDDomain defines an admissible continuous domain (or phase) in the hybrid system model with Hybrid Zero Dynamics family of controllers.

Contraints are typically given as kinematic constraints, such as holonomic constraints and unilateral constraints, of the rigid body model.
