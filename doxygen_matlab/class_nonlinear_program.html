function obj = NonlinearProgram.genConstrIndices ( char  solver)

This function generates the indexing information for constraints.

Note
Indexing NLP constraints depends on different NLP solver being used. By default, we assume ipopt being used, with which all constraints are catagorized into one big group. If fmincon being used, then there will be linear/nonlinear and equality/inequality constraints, and each group of constraints are indexed separately.

function [ NlpConstr constrArray , struct constrInfos ] = NonlinearProgram.getConstrInfosIpopt ( )

This function returns the indexing information of the constraints function array, including the Gradient and Hessian (if applicable) sparse structures.

Return values
constrArrayAn array of constraints
constrInfosA structure contains other pre-computed information regarding constraints array.
Required fields of constrInfos:
  • dimsConstr —  The dimension of constraints
  • cl —  The lower bound of constraints
  • cu —  The upper bound of constraints
  • nnzJac —  The number of nonzero entries in the objective gradient vector
  • gradSparseIndices —  row and column indices of the nonzero entries in the sparse Gradient vector
  • nnzHess —  The number of nonzero entries in the cost portion of the Hessian
  • hessSparseIndices —  row and column indices of the nonzero entries in the sparse Hessian matrix associated with the cost function
