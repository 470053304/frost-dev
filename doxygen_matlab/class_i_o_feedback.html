function obj = IOFeedback.setParam (   varargin)
virtual

Sets the controller parameters.

Note
When using 'ep' as the control parameters, we use the following relationships for 'kp' and 'kd': \(kp = ep^2\), \(kd = 2*ep\).
Please do not specifiy 'ep' and ('kp','kd') at the same time. In this case, only 'ep' will be used.
Parameters
vararginname-value pairs of control parameters. Expected arguments are:
setParam ( ep, kp, kd )
Required Parameters for varargin:
  • ep epsilon for human-inspired control
  • kp kp
